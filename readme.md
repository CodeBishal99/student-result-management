# Student Result Management System

A simple Python project that manages student records using SQLite database.

## Features

- Add student
- View all students
- Search student by roll number
- Delete student
- SQLite database integration

## Technologies Used

- Python
- SQLite

## Project Structure

```text
student-result-management/
│
├── main.py
├── database.py
├── students.db
├── README.md
└── requirements.txt
```

## How to Run

1. Clone repository

```bash
git clone https://github.com/yourusername/student-result-management.git
```

2. Open project folder

```bash
cd student-result-management
```

3. Run project

```bash
python main.py
```
